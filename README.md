# Backtesting-VaR-and-Expected-Shortfall-EWMA-vs-GARCH-for-the-S-P-500
Compared EWMA and GARCH(1,1) volatility models under Normal and t-Student distributions for VaR and ES estimation on S&amp;P 500 weekly returns. Applied 4 formal backtesting procedures (Kupiec, Christoffersen, Ch-P, McNeil-Frey) across 752 out-of-sample weeks. GARCH t-Student was the sole specification
